# Corks 官网

这是 Corks 社交饮酒 App 的官方网站项目。该网站旨在展示 Corks App 的核心功能、品牌理念以及未来发展计划。

## 技术栈

- Next.js 13 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- next-i18next

## 功能特点

- 响应式设计
- 中英文双语支持
- 现代化动画效果
- SEO 优化
- 性能优化

## 开始使用

1. 克隆项目

```bash
git clone [repository-url]
```

2. 安装依赖

```bash
npm install
```

3. 运行开发服务器

```bash
npm run dev
```

4. 构建生产版本

```bash
npm run build
```

## 项目结构

```
src/
  ├── app/              # App Router 页面
  ├── components/       # 可复用组件
  ├── styles/          # 全局样式
  └── utils/           # 工具函数
```

## 部署

项目可以部署到 Vercel 平台，支持自动部署和持续集成。
